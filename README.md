# Exercicio_Programacao_Orientada_Objetos_Heranca_Sobrescrita
Implementação do exercício proposto pelo professor Archimedes Detoni, como uma avaliação prática da disciplina de POO. O objetivo do exercício é explorar os conceitos de herança e sobrescrita de métodos em classes, aplicando princípios fundamentais da orientação a objetos como reutilização de código e especialização de classes.
